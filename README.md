# 推荐模块1：打造炫酷轮播及列表
## 轮播组件开发
- `Recommend` 组件

- `Slider` 组件
    -   安装 `swiper` 插件
    ```bash
    yarn add swiper
    ```

    - 引入 CSS
    ```js
    // slider/index.js
    import "swiper/swiper.min.css";
    ```

## 分布拆解 `scroll` 组件
- 安装 `better-scroll`
```bash
yarn add better-scroll@next
```

## axios 请求封装
- 安装 
```bash
yarn add axios
```
